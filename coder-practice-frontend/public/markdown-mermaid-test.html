# 算法流程图示例

## 数组求和算法流程

\`\`\`mermaid
graph TD
    A[开始] --> B[初始化sum为0]
    B --> C{还有元素吗?}
    C -->|是| D[将当前元素加到sum]
    D --> C
    C -->|否| E[返回sum]
    E --> F[结束]
\`\`\`

## 复杂流程图

\`\`\`mermaid
flowchart TD
    A[开始] --> B{用户登录?}
    B -->|已登录| C[显示主页]
    B -->|未登录| D[跳转登录页]
    C --> E[用户操作]
    D --> F[用户登录]
    F --> B
    E --> G[结束]
\`\`\`
